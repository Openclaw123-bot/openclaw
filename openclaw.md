## 二、Win10 安装前「致命避坑」（不看必部署失败）
这是保障 Win10 系统顺利完成部署的核心环节，请严格按照要求执行：
1. 彻底关闭各类安全防护软件：包括 360 安全卫士、360 杀毒、腾讯电脑管家、火绒以及 Win10 自带的微软 Defender 实时防护。OpenClaw 需要控制系统、读写本地文件，容易被安全软件误判，进而删除核心文件，造成部署失败。
2. 关闭 Win10 文件保护功能：右键「此电脑」→「属性」→「高级系统设置」→「系统保护」，临时关闭安装磁盘的系统保护，防止系统阻止文件写入。
3. 预留足够磁盘空间：目标磁盘需保证 4G 及以上空闲空间。优先选择 D 盘、E 盘进行安装，Win10 C 盘权限管控严格，极易出现安装异常，不建议选用。
4. 脱离压缩文件夹运行：不要直接在 Win10 压缩文件夹内启动程序，会引发文件读取故障，务必完整解压后再进行后续操作。

Win10 默认开启 SmartScreen 安全保护机制，会拦截未签名的应用程序，这也是该系统用户部署时最常遇到的问题，按照以下步骤即可正常启动：
1. 打开解压后的`Openclaw-win`文件夹，找到「Openclaw Windows 一键启动.exe」（红色龙虾图标）并双击运行。
2. 此时会弹出「Windows 已保护你的电脑」提示窗口，这是 Win10 常规拦截机制，并非程序存在风险。
3. 点击窗口左下角「更多信息」，展开选项后点击「仍要运行」，即可启动专属安装程序。

✅ 实用技巧：如果想要后续不再出现该拦截提示，可右键程序→「属性」→「常规」→「解除锁定」，点击确定即可。

## 六、第四步：Win10 专属部署配置（路径 / 权限 / 安装）
程序启动后全程采用可视化操作，搭配 Win10 专属优化配置，零基础也能顺利完成设置。

### 1. 进入初始化界面
程序启动后会弹出带有红色龙虾 Logo 的 OpenClaw 欢迎界面，点击底部红色「开始使用」，进入安装配置页面。

### 2. 选择合规安装路径（关键步骤）
Win10 系统对包含中文、特殊字符、空格的文件路径权限限制严格，很容易导致部署失败，路径设置要求如下：
- 必须选用纯英文路径，路径中不得出现中文、空格、￥、& 等特殊字符
- 推荐路径：`D:\OpenClaw`、`E:\AI\OpenClaw`，这类路径适配 Win10 权限规则，稳定易用
- 错误路径示例：`D:\软件\小龙虾`（含中文）、`C:\Program Files\OpenClaw`（含空格）、`D:\OpenClaw·2.7.9`（含特殊字符）

额外检查：确认所选磁盘未开启加密、无权限锁定，空闲空间不低于 4G。

### 3. 确认安装协议
勾选「我已阅读并同意《用户协议》和《免责声明》」，随后点击「开始安装」，程序会自动适配 Win10 系统执行部署操作。

### 4. 等待自动部署
点击安装按钮后，全程耗时 3-5 分钟，具体时长由电脑配置决定。等待过程中不要关闭窗口、不要操作电脑。程序会自动完成以下工作：
1. 检测 Win10 系统环境，补充缺失的运行依赖
2. 部署 OpenClaw 核心文件，适配 Win10 注册表
3. 安装适配 Win10 的浏览器控制、键鼠模拟工具

## 八、第六步：Win10 系统专属常见问题与解决办法
### Q1：安装时提示「权限不足，无法写入文件」？
A：该问题由 Win10 磁盘权限引发，可按顺序尝试解决：
① 右键「Openclaw Windows 一键启动.exe」，选择「以管理员身份运行」；
② 将安装路径更换至 D 盘或 E 盘；
③ 关闭 Win10 用户账户控制（UAC），重启电脑后重新安装。

### Q2：启动后 Gateway 一直离线，重启程序也无法解决？
A：
① 确认所有安全软件、Defender 实时防护均已关闭，恢复被隔离的`Openclaw-win`文件夹内全部文件；
② 检查安装路径，确保为纯英文，无中文、特殊字符；
③ 右键桌面 OpenClaw 快捷方式→「属性」→「兼容性」，勾选「以兼容模式运行这个程序（Win10）」以及「以管理员身份运行此程序」，保存设置后重启程序。

### Q3：AI 无法打开软件、操控键鼠？
A：OpenClaw 需要获取 Win10 键鼠操控权限：
① 打开 Win10「设置」→「隐私和安全性」→「权限」，开启「应用访问键鼠」「文件访问」权限；
② 关闭系统护眼模式、分屏工具，避免界面遮挡；
③ 以管理员身份重启 OpenClaw。

### Q4：第一次启动卡在「加载中」，时长超过 3 分钟？
A：主要因系统服务加载缓慢导致：
① 关闭微信、浏览器等后台程序，降低内存占用；
② 直接关闭程序，以管理员身份重新运行；
③ 检查网络状态，第一次启动需要少量网络完成初始化，断网会造成加载失败。

### Q5：桌面没有生成 OpenClaw 快捷方式？
A：
① 进入解压后的`Openclaw-win`文件夹，右键「Openclaw Windows 一键启动.exe」，选择「发送到」→「桌面快捷方式」；
② 重新运行安装程序，选择「修复安装」，程序会重新创建桌面快捷方式。
