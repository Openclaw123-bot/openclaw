## 前言📖
OpenClaw（小龙虾）是一款主打本地运行的开源AI自动化智能体，凭借可视化操作、全流程自动化、电脑操控能力，成为热门的桌面数字员工工具。

区别于普通对话AI，OpenClaw可以直接操作系统资源，通过自然语言指令自动完成文件整理、数据处理、桌面自动化等办公任务。
本篇教程适配 Windows、Mac 双平台，采用整合式一键安装包，全程无需代码、无需配置环境，零基础用户也能快速部署完成。

## 一、安装前置关键注意事项⚠️
在解压、安装、启动程序前，请务必关闭本机所有安全防护软件：
Windows Defender、火绒、360安全卫士、腾讯电脑管家等（含后台进程）。

OpenClaw需要读写文件、模拟键鼠、控制系统窗口，容易被安全软件误拦截，导致核心文件丢失、部署失败。
该项目为开源项目，源码可公开核验，无安全风险。

## 三、正确解压与程序启动步骤✅
1. 推荐使用 7-Zip、WinRAR 解压，**不使用系统自带解压工具**
2. 将压缩包解压至自定义文件夹
3. 进入目录，双击运行：`Openclaw Windows 一键启动.exe`
4. 遇到 Windows SmartScreen 拦截，点击「更多信息」→「仍要运行」

## 四、安装配置与全自动部署⚙️
1. 打开程序后点击底部「开始使用」进入安装界面
2. **安装路径必须为纯英文**，禁止中文、空格、特殊字符
- ✅ 推荐路径：D:\OpenClaw、E:\AI\OpenClaw
- ❌ 禁止路径：含中文、空格、汉字文件夹

3. 勾选用户协议，点击「开始安装」
4. 程序自动完成环境检测、依赖部署、配置生成、快捷方式创建

部署全程 3–5 分钟，请勿关闭窗口，避免部署中断。

## 五、服务初始化与部署成功校验💡
安装完成后程序自动启动，第一次启动会初始化 Gateway 服务。

提示「正在等待 Gateway 就绪...」属于正常加载过程，等待 1~3 分钟即可。

**右上角显示【Gateway 在线】即部署成功**，可以正常使用AI自动化能力。

## 七、常见报错快速解决方案🛠️
### 1. 安装无响应、部署失败
关闭全部安全软件后台，重新解压安装包重试。

### 2. Gateway 一直离线
检查路径是否纯英文 → 重启网关服务 → 管理员身份运行程序。

### 3. 启动提示网络异常
关闭代理、VPN，保持网络通畅，等待依赖初始化完成。

### 4. 无法发送指令
等待网关完全在线，若异常持续可重新部署。

3. 支持接入微信、飞书等渠道，可实现远程控制下发任务
4. 版本升级直接覆盖安装，无需卸载旧版本
