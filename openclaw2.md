- 覆盖多元场景：支持文件分类整理、表格制作、网页浏览、数据统计、邮件收发等办公操作。

## 二、安装前期准备工作
在下载、解压以及启动程序之前，请务必关闭电脑中 360 安全卫士、腾讯电脑管家、火绒等所有安全防护软件。

OpenClaw 拥有键鼠模拟、文件读写、调用系统权限等功能，容易被安全软件误判为风险程序，进而出现拦截进程、删除核心文件的情况，最终导致安装失败。该项目为正规开源项目，大家可前往 GitHub 查验相关资质，关闭防护软件后再继续后续步骤即可。

### 2. 规范解压文件
不推荐使用 Windows 系统自带解压工具，极易造成文件缺失、损坏，优先选用 WinRAR、7-Zip 等专业解压软件。

## 四、启动程序并处理系统拦截
### 1. 打开启动程序
进入解压完成的 `Openclaw-win` 文件夹，找到 **Openclaw Windows 一键启动.exe**，凭借红色龙虾图标可快速识别，双击运行该程序。

### 2. 应对系统安全弹窗
部分 Windows 系统会弹出安全保护提示，这是系统 SmartScreen 的常规防护拦截，并非程序存在安全问题。

处理方式：点击弹窗内「更多信息」，再选择「仍要运行」，即可正常进入安装流程。若无相关弹窗，可直接进行下一步操作。

## 五、全自动安装部署流程
### 1. 进入安装配置界面
程序启动后会展示带有龙虾标识的欢迎页面，点击底部「开始使用」按钮，进入安装配置环节。

### 2. 设置合规安装路径
安装路径必须使用**纯英文**，路径内禁止出现中文、空格、特殊符号，路径格式不规范会直接导致安装终止。
- 不建议将软件安装在 C 盘，避免占用系统盘空间，影响电脑整体运行速度；
- 推荐路径参考：`D:\OpenClaw`

避坑提示：切勿使用包含中文名称的文件夹作为安装目录

选定路径后，勾选同意用户协议与免责声明，点击「开始安装」。

### 3. 等待自动部署完成
点击安装按钮后，程序会自动完成环境检测、依赖组件补全、核心文件部署、浏览器插件安装、桌面快捷方式创建等一系列操作，全程无需手动操作。整体耗时 3-5 分钟，具体时长由电脑硬件配置决定。

> ⚠️ 重要提醒：部署过程中不要关闭安装窗口，一旦操作中断，需要重新执行完整安装流程。

### 4. 第一次启动加载说明
安装完成后程序会自动开启主界面。第一次启动时，Gateway 服务需要完成初始化加载，页面会短暂处于加载状态，耐心等待 1-3 分钟即可正常使用。后续再次启动，加载速度会大幅提升。

## 七、常见问题排查与解决方案
**Q1：文件被安全软件隔离、删除该如何处理？**
A：彻底关闭所有安全软件及其后台进程，从隔离区恢复相关文件，重新解压安装包并启动程序完成部署。

**Q2：安装时提示路径格式错误，无法继续安装？**
A：重新设置纯英文安装路径，清除路径内所有中文、空格与特殊符号，再次点击安装即可。

**Q3：Gateway 一直显示离线，无法下发任务？**
A：先检查安全软件是否完全关闭、安装路径是否合规；点击界面重启按钮重置服务，若问题依旧，重新运行启动程序进行部署。

**Q4：第一次启动加载速度偏慢？**
A：该现象属于正常情况，第一次启动需要加载全部运行组件，等待片刻即可，后续启动会更加流畅。
