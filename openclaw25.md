# OpenClaw v2.7.9一体化部署教程，快速搭建本地AI自动化办公工具
适配系统：Windows10/11 64位
当前版本：v2.7.9（虾壳云版）
工具亮点：全可视化操作界面，摒弃命令行操作，无需手动配置Python、Node.js运行环境，所有运行依赖全部内置整合，整套部署流程仅需五分钟，零基础使用者也能独立完成搭建。


## 安装前置重要提醒（规避各类部署故障）
在解压、安装、启动程序之前，需要彻底关闭电脑全部安全防护程序，包括360安全卫士、腾讯电脑管家、火绒安全、Windows Defender实时防护功能。

该工具具备本地文件读写、键鼠模拟、系统操控等自动化能力，极易被安全软件判定为风险程序，一旦核心文件被拦截删除，会直接导致安装流程中断。项目为开源项目，可前往GitHub核验程序安全相关内容，关闭防护工具后再执行安装操作。

## 第一步：解压安装包并启动程序，设置合规安装目录
推荐使用WinRAR、7-Zip两款解压工具，系统自带解压程序容易造成文件损坏；右键压缩包选择解压至当前文件夹，解压完成后会生成Openclaw-win文件夹。

打开文件夹，双击带有红色龙虾标识的「Openclaw Windows 一键启动.exe」；若弹出Windows安全拦截弹窗，点击更多信息，选择仍要运行即可解除拦截。

程序启动后跳转欢迎界面，点击页面底部开始使用，进入安装目录选择页面。

### 路径硬性规范
安装路径仅支持纯英文，不能包含中文、空格、特殊符号，不建议安装在C盘占用系统存储空间。
- 推荐安装路径：D:\\OpenClaw、E:\\AI\\OpenClaw
- 不合规路径示例：D:\\工具\\OpenClaw、D:\\小龙虾

勾选用户协议与免责声明，点击开始安装进入自动部署阶段。

## 第二步：等待全自动部署，全程无需人工干预
点击开始安装后，程序自动执行全部部署工作，整套流程耗时3-5分钟，时长由电脑硬件配置决定，部署过程中不要关闭窗口，否则进程直接中断。

程序自动完成的工作：
1. 扫描本机运行环境，自动补齐Git、Python、Node.js等缺失依赖组件
2. 部署适配Windows系统的OpenClaw核心程序文件
3. 安装浏览器控制、键鼠模拟配套组件，支撑各类自动化操作
4. 生成专属本地配置文件，完成初始化并生成桌面快捷启动图标

进度条达到100%后，软件主界面会自动弹出。

## 第三步：第一次启动软件，核验后台服务运行状态
安装结束自动进入初始化页面，页面提示正在等待Gateway就绪。第一次启动需要完成后台服务初始化，等待1-3分钟，后续启动仅需数秒，属于正常运行现象。

界面右上角显示Gateway在线，即代表部署全部完成，各区域功能说明：
1. 右上角：Gateway运行状态、服务重启按钮、日志查看入口、可用Tokens总量
2. 左侧：本地对话、渠道切换栏，支持新建对话、查看历史交互记录
3. 中间：AI对话窗口，内置默认智能助手，支持多模型切换、代码自动高亮
4. 底部：自然语言指令输入框，回车发送指令，Shift+Enter换行，支持两种运行模式切换

## 第四步：下发AI指令，体验电脑自动化操作
部署完成无需额外配置，直接在底部输入框输入自然语言指令，软件会自动拆解任务、调用工具、操控电脑执行操作，指令描述越细致，执行效果越贴合需求。

新手可直接复制测试指令体验功能：
1. 整理D盘下载文件夹，按照文件类型新建分类文件夹归档
2. 打开记事本，录入OpenClaw部署完成字样，保存文件至桌面
3. 读取电脑各磁盘剩余可用空间，整理文字反馈结果

## 常见故障快速解决办法
1. \*\*安装失败、启动无响应\*\*：确认安全软件全部关闭，删除原有解压文件，重新解压安装包重试
2. \*\*Gateway持续离线\*\*：更换纯英文安装路径、点击重启服务、右键以管理员身份运行程序
3. \*\*网络异常提示\*\*：保证网络正常连接，关闭代理、VPN工具后重启软件
4. \*\*Tokens额度不足\*\*：可在界面充值入口补充额度，不会影响核心自动化功能使用
5. \*\*无法输入、发送指令\*\*：等待Gateway在线后再操作，异常情况可重装程序

## 实用操作建议
1. 优先选择剩余存储空间5G以上磁盘分区安装，预留空间存放模型缓存、拓展插件
2. 桌面快捷方式生成后，后续直接双击图标启动，无需重复解压安装包
3. 想要对接聊天渠道，部署完成后进入设置-聊天渠道完成配置，实现在聊天界面下发任务
4. 版本更新无需卸载旧程序，直接下载新版安装包覆盖原有文件夹即可
