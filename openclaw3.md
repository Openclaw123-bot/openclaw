## 一、认识 OpenClaw（小龙虾）：功能与核心亮点
不少人会将 OpenClaw 当作普通对话 AI，实际上它是一款可以直接操控设备的智能数字员工。它能够识别自然语言指令，自动拆分任务、调用各类工具并完成全流程操作，全程无需人工介入。

因其红色龙虾 logo 以及 “用钳子承接各类任务” 的寓意，圈内用户习惯将部署和调试这款软件称作 “养虾”，作为热门开源项目，它的综合表现获得了大量用户认可。

### 核心亮点
1. **本地运行模式**：所有数据都保存在本地设备中，有效保护隐私，杜绝敏感信息外泄。
2. **低使用门槛**：不用掌握编程知识，也无需手动输入命令，一键即可完成部署，零基础用户也能快速上手。
3. **多端适配**：兼容 Windows、Mac、Linux、安卓、苹果等主流系统，同时支持对接微信、飞书、Slack 等软件，随时随地下发操作指令。
4. **即装即用**：部署包内置全部运行依赖组件与基础功能，解压之后就能正常使用。
5. **办公能力全面**：可完成文件整理、邮件发送、表格制作、浏览器自动化、数据统计处理等多种办公场景操作。

## 二、部署前置提醒 规避常见问题
在下载解压、启动程序之前，请彻底关闭 360 安全卫士、腾讯电脑管家、火绒等所有安全防护软件。

补充说明：OpenClaw 需要获取系统操作权限，实现文件读写、键鼠模拟等功能，很容易被安全软件误判为风险程序，进而被拦截或删除核心文件，导致部署失败。关闭防护软件后再进行安装操作即可正常运行，项目开源信息可前往 GitHub 进行核验。

## 四、第二步：启动主程序 处理系统拦截
### 1. 进入解压目录
打开刚刚解压生成的 Openclaw-win 文件夹，找到主程序 Openclaw Windows 一键启动.exe，程序图标为红色龙虾样式。

### 2. 解决系统 SmartScreen 拦截
双击启动程序，部分 Windows 设备会弹出系统防护提示，这是系统常规安全拦截，并非程序存在安全问题。

处理方式：点击弹窗中的「更多信息」，接着选择「仍要运行」，就能正常启动安装程序。

## 五、第三步：全自动部署配置
### 1. 进入初始化界面
程序启动后会弹出欢迎页面，点击页面底部红色「开始使用」按钮，进入安装配置界面。

### 2. 规范设置安装路径（重点）
安装路径只可使用纯英文，路径内不能出现中文、空格、特殊符号，否则部署会直接失败。
建议不要将软件安装在 C 盘，避免占用系统盘空间，影响设备运行效率。

推荐路径示例：`D:\OpenClaw`
错误路径示例：`D:\ 办公软件 \OpenClaw`、`D:\ 小龙虾`

选定路径后，勾选同意用户协议与免责声明，点击「开始安装」。

### 3. 等待自动部署流程
点击安装按钮后，程序会全自动执行部署工作，全程无需手动操作，根据电脑配置不同，耗时大约 3-5 分钟。

部署包含这些内容：
- 检测并补齐系统运行依赖
- 部署 OpenClaw 2.7.9 核心文件
- 安装浏览器自动化组件
- 生成系统适配配置
- 创建桌面快捷方式

> ⚠️ 注意：部署过程中请勿关闭窗口，否则会造成安装中断，需要重新解压安装。

### 4. 首次启动加载说明
安装完成后软件会自动开启主界面，首次启动时 Gateway 服务需要初始化，页面会显示加载状态，耐心等待 1-3 分钟即可正常进入聊天操作界面，该情况属于正常现象。

## 七、常见问题排查方案
### Q1 启动被安全软件拦截，核心文件被删除
先彻底关闭安全软件后台进程，重新解压安装包并启动程序。如果文件已经被隔离，在安全软件隔离区恢复 Openclaw-win 内全部文件，再重新执行部署操作。

### Q2 安装提示路径含中文 / 特殊字符
重新修改安装目录，保证路径全为英文字符，剔除中文、空格与特殊符号，再次点击安装即可。

### Q3 Gateway 持续离线，无法发送指令
1. 检查安全软件是否完全关闭、安装路径是否合规；
2. 点击界面右上角重启按钮，重启 Gateway 服务；
3. 以上操作无效时，关闭软件，重新运行一键启动程序重装部署。

### Q4 首次启动加载时间过长
首次启动需要初始化各类依赖组件，等待 1-3 分钟属于正常情况，后续再次启动速度会明显提升。
