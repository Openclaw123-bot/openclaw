# Win10/11本地AI自动化部署｜OpenClaw 2.7.9零基础可视化安装教程
## 一、工具简介
OpenClaw（小龙虾）是一款主打**本地运行、可视化、零代码**的桌面AI自动化智能体，可直接操控电脑完成文件整理、网页操作、数据提取、办公批量任务等操作。相比传统AI对话工具，OpenClaw具备真实电脑操控能力，无需人工一步步干预，输入自然语言即可自动拆解并执行整套流程。

本文基于最新 **OpenClaw 2.7.9** 版本整理完整部署流程，全程不敲代码、不配置复杂环境，新手也能快速完成本地搭建。

## 二、核心功能亮点
- **本地运行保护隐私**：数据全部留存本机，不强制上传云端，安全性更高
- **全可视化操作**：图形化界面，无需命令行、无需编程基础
- **全自动环境适配**：自动补齐 Node、Git、Python 依赖
- **多系统、多渠道兼容**：支持 Windows、Mac，可对接微信、飞书远程控机
- **办公场景全覆盖**：文件分类、网页自动化、表格汇总、资料批量整理

## 三、安装前置注意事项（必看）
部署失败90%来源于安全软件拦截，安装前请务必：

**关闭所有安全防护**
- 360安全卫士、腾讯电脑管家、火绒安全
- Windows Defender 实时防护

OpenClaw需要读写本地文件、模拟键鼠、调用系统进程，容易被误报风险。项目为开源项目，可自行核验源码安全，关闭防护仅用于避免误删文件。

## 五、解压流程规范

## 十、常见问题排查
### 1. 安装失败、文件缺失
关闭所有杀毒 → 重新解压安装包重试

### 2. Gateway 一直离线
- 检查路径是否全英文
- 重启网关服务
- 管理员身份重启程序

### 3. 启动加载慢
第一次启动需要初始化环境，等待片刻即可，后续秒开

### 4. 无法发送指令
等待网关完全在线后再进行操作
